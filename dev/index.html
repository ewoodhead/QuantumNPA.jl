QuantumNPA.jl

Maximise CHSH:

julia> using QuantumNPA

julia> @dichotomic A1 A2 B1 B2;

julia> S = A1*(B1 + B2) + A2*(B1 - B2);

julia> npa_max(S, 2)
2.828227712681755
